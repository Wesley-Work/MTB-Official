# 🎉 About

- 媒体部官网代码仓库，使用Vue3+Vite6开发，后端为Python
- 支持通过OA([MTB-OA](https://github.com/Wesley-0808/MTB-OA))修改页面内容
- ShareNetdisk页面使用了『[TDesign](https://github.com/Tencent/Tdesign-vue-next)』组件库
- 顺德中专团委学生会媒体部版权所有

# 🔨 Develop

```shell
# Install
pnpm i

# Test
pnpm run dev

# Build
pnpm run build
```
